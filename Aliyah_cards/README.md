# Aliyah Cards and Box to hold them
Everything in this folder was initially laser cut, but it should be fine to 3d print if that is easier for you. 
You may want to print the box in one piece if so. Also, the word "עליות" is cut through the model, but I only intended to raster it into the surface. 
You will need to fill that in if you do not want holes in the box.

I cut the box into MDF and friction fit the pieces together. 
I later changed this to an acrylic cut glued together with glass-glue since I thought it looked better.

The box exactly fits 17 cards when laser cut.

I recomend cutting the lid in a trasparent material.
